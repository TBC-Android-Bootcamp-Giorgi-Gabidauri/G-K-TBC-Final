# G-K-TBC-Final

Purpose
-----------
- This app has been creaeted in purpose to make users easier to buy and sell products.

Functionality
--------------
in this app you can:
- easily search for products you want to buy.
- this app has bunch of categories for you to search.
- you can filter products by seller type, condition, price, and location.
- you can save products you'd like to see later.
- user has in-app wallet where he can easily fill the balance and see transactions.
- here you can sell products easily and make users easy to find it by choosing correct information.
- you can monitor active and sold products.
- when product is sold, you get a notification.
- you can see your information on user profile and edit it.
